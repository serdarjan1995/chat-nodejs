## Chat app with socket.io
#### written with nodejs


----

### How to run  
```
npm install
```
```
npm run start
```

----

### Building with docker
```
sudo docker-compose up -d --build
```
Access via `http://localhost`  


_Note: If you wish change nginx expose port to any.  
So that you can access `http://localhost:YOUR_PORT`)_
